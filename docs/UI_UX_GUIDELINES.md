# UI / UX Design Guidelines

Project: Điện Máy Nga Sơn

Version: MVP 1.0

---

# 1. Design Philosophy

This website is NOT:

* Apple Website
* SaaS Dashboard
* Startup Landing Page
* Enterprise Software

This website IS:

* Local Home Appliance Store
* Local Repair Service Website
* Lead Generation Website

Primary Goal:

Convert visitors into:

* Phone Calls
* Zalo Messages
* Facebook Messages
* Quote Requests

Every page should maximize conversion.

---

# 2. Design Inspiration

Combine:

1. Điện Máy Xanh
2. HVAC Service Websites
3. Modern SaaS Visual Design

Learn from Điện Máy Xanh:

* Product cards
* Product listing
* Product detail layout
* Product information hierarchy

Learn from HVAC websites:

* CTA placement
* Lead forms
* Service landing pages
* Trust sections

Learn from SaaS:

* Spacing
* Typography
* Clean layouts
* Component consistency

---

# 3. Brand Personality

Brand Values:

* Trustworthy
* Friendly
* Professional
* Local
* Fast Response

Avoid:

* Luxury
* Corporate
* Experimental
* Futuristic

---

# 23. Accessibility

Use:

Proper labels

Keyboard navigation

Contrast compliance

---

# 24. Performance

Images:

Next/Image

Lazy Loading

Responsive Images

Avoid:

Heavy Animations

Large Videos

Background Videos

---

# 25. Animation Rules

Use:

Fade

Slide

Scale

Duration:

150ms - 300ms

Avoid:

Complex animations

Parallax

3D effects

---

# 26. Conversion Rules

Every important page must contain:

Phone CTA

Zalo CTA

Facebook CTA

Quote CTA

Never leave users without an action.

---

# 27. Design Anti-Patterns

Do NOT use:

Auto-playing sliders

Full-screen popups

Intrusive modals

Excessive gradients

Glassmorphism

Dark-only design

Cryptocurrency style UI

Startup landing page layouts
