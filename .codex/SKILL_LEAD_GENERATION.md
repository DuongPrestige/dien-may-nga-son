# SKILL_LEAD_GENERATION.md

# Lead Generation Skill

Project: Điện Máy Nga Sơn

Purpose:

Teach AI agents how to maximize lead generation.

This website is not an e-commerce website.

The goal is not checkout.

The goal is lead conversion.

---

# Lead Hierarchy

Primary Conversion:

1. Phone Call

Secondary Conversion:

2. Zalo Message

Third Conversion:

3. Facebook Message

Fourth Conversion:

4. Quote Form Submission

Always optimize for these actions.

---

# Core Principle

Every important page must answer:

"What should the user do next?"

If there is no obvious CTA:

The page is incomplete.

---

# Homepage Conversion Rules

Homepage must contain:

Hero CTA

Service CTA

Product CTA

Sticky CTA

Contact CTA

Footer CTA

Homepage should never require scrolling to find contact information.

---

# Product Page Conversion Rules

Every product page must contain:

Call Button

Zalo Button

Facebook Button

Quote Form

Price

Price Notice:

"Liên hệ để nhận báo giá chính xác và giá tốt hơn."

Related Products

FAQ

Never leave product pages without conversion paths.

---

# Service Page Conversion Rules

Service pages are high-conversion pages.

Each service page must include:

Hero CTA

Problem Section

Trust Section

CTA Section

FAQ Section

Quote Form

Final CTA Section

---

# Contact Rules

Contact information must be visible.

Display:

Phone

Zalo

Facebook

Address

Working Hours

Do not hide contact details.

---

# Mobile Conversion Rules

Mobile is highest priority.

Must have sticky bottom CTA.

Buttons:

Call

Zalo

Facebook

Quote

Visible on all pages.

---

# CTA Copy Rules

Use direct language.

Good:

* Gọi ngay để được tư vấn
* Nhắn Zalo nhận báo giá
* Liên hệ Facebook
* Gửi yêu cầu báo giá

Avoid vague CTAs:

* Learn More
* Discover
* Explore

---

# Trust Building Rules

Before asking for a lead:

Provide trust.

Use:

* Genuine products
* Warranty
* Experience
* Local service
* Testimonials
* Fast support

---

# Lead Form Rules

Keep forms short.

Required:

Name

Phone

Optional:

Message

Never ask for:

* Date of birth
* Address
* Too many fields

Short forms convert better.

---

# Lead Tracking Rules

Store:

Name

Phone

Message

Source URL

Source Type

Product ID

Service ID

Created Date

Status

# Lead Source Types

# CTA Placement Rules

Place CTA:

Above the fold

Middle of content

End of content

Important pages should have multiple CTAs.

---

# Phone CTA Rules

Phone CTA is highest priority.

Desktop:

Header

Sidebar

Footer

Mobile:

Sticky Bottom CTA

---

# Zalo Rules

Zalo is second highest priority.

Always visible.

Use:

Open Chat

Not hidden behind menus.

---

# Facebook Rules

Facebook is optional support channel.

Lower priority than:

Phone

Zalo

---

# Form Success Rules

After successful submission:

Show success message.

Example:

"Cảm ơn bạn. Điện Máy Nga Sơn sẽ liên hệ với bạn trong thời gian sớm nhất."

---

# Lead Dashboard Rules

Admin dashboard should highlight:

New Leads

Recent Leads

Lead Status

Quick Contact Information

---

# Lead Optimization Rules

If a design choice conflicts with lead generation:

Choose lead generation.

Examples:

* Simpler forms
* More visible CTA
* More contact information
* Faster access to phone/Zalo

---

# Success Metrics

Track:

Phone Clicks

Zalo Clicks

Facebook Clicks

Lead Form Submissions

Lead Conversion Rate

Not:

Time On Site

Fancy engagement metrics
